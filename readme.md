# Soundscape Project 

- Project Description 

    Soundscape is a Spotify-Inspired web music player built using HTML, CSS, Javascript
    The application allows users to browse songs, albums, artists, and favorites while 
    providing an interactive music playback experience.


- How to run application

    You can run the application using the Live Server Extension in VS Code, 
    You can also run it in this live deployment if you go to this url 
    https://soundscape-project-zeta.vercel.app/

- Features Implemented

    The features in this project are 
        - Users can start and stop the song directly from the player interface
        - It also includes a skip navigation
        - Music Library Navigation
        - Favorite functionality when you click the heart in the player it will automatically add in the favorites
        - Searchbar, You can search a song here
        - Theme toggle
    
- Known Limitation or Bugs

    - The music library is currently limited to around 20 songs only.
    - The application is not fully responsive on smaller devices such as phones, iPhones, and tablets
    - Favorite songs are stored temporarily in memory only and are not saved after refreshing the page.
    - The project currently has no backend or database integration.
    - Some UI elements may overflow or require scrolling on smaller screen sizes.
    - The music player does not yet support playlist creation or queue management.
    - Search functionality is limited to basic filtering only.

- Design Decisions and HCI principles applied 
    - The application uses a consistent color palette, typography, and layout inspired by modern music streaming platforms such as Spotify. Consistent button styles, spacing, and navigation layouts help users understand the interface more easily.

    - Basic readability and spacing were considered to improve user interaction. Larger clickable areas and visible icons help users interact with controls more comfortably.

    - The interface was designed for users who want a lightweight and simple music player experience.
